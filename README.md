# n8n-japan-rpa-lite

日本の中小企業・自治体向けに、**IT人材不足でも業務自動化を実現できるRPAライトOSS**を開発します。  
OSSのワークフロー自動化プラットフォーム **n8n** をベースに、**Zapierライクな日本仕様ツール**として進化させます。

---

## 🎯 プロジェクトの目的

- IT人材不足でも日常業務の自動化を可能にする
- 英語UI・高額ツール依存から脱却し、OSSでコストゼロ化
- 日本独自のSaaS（freee / kintone / BASE など）に特化
- 自治体・商工会議所DXでも安心して使える国内OSS環境を整備

---

## 🛠 解決したい課題

- **中小企業**  
  - IT担当が不在でSaaSを活用できない  
  - 請求書作成や顧客通知など定型作業に時間を取られている  

- **自治体**  
  - データ連携・オープンデータ活用が進まない  
  - 海外クラウド依存ツールはセキュリティ要件で導入不可  

- **OSSコミュニティ**  
  - 日本SaaSに対応した自動化OSSが存在しない  
  - 英語ベースのn8nは導入障壁が高い  

---

## 🚀 初期機能案

- **日本語対応**  
  - GUIの完全日本語化  
  - 日本語チュートリアル / ドキュメント / FAQ  

- **日本SaaS対応ノード追加（必須）**  
  - 会計系：freee / マネーフォワード / 弥生会計  
  - CRM系：kintone / Sansan  
  - EC系：BASE / 楽天市場 / Yahoo!ショッピング  

- **自動化テンプレート（導入の入口となる業務）**  
  - **請求書発行自動化**  
    - Googleスプレッドシート → PDF請求書生成 → Gmail送付  
  - **EC受注処理自動化**  
    - BASE / 楽天市場の注文 → freee会計に自動登録 → Slack通知  
  - **自治体・商工会議所向け通知自動化**  
    - Googleフォーム回答や申請データ → kintone登録 → 担当者にメール通知  

- **ウィザード型GUI**  
  - 「どんな業務を自動化したいですか？」と質問に答えるだけでフロー生成  
  - 初心者でも **数クリックで業務自動化** が可能  

---

## 🏗 技術スタック（予定）

- **ワークフローエンジン**: n8n  
- **GUI強化**: Electron + React  
- **Web自動操作**: Playwright  
- **データ可視化**: Streamlit / Dash  
- **AI統合（将来）**: OpenAI API（問い合わせ要約や分類）
